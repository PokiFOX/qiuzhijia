## 一、接口概览

| 接口类别 | 接口路径 | 方法 | 说明 |
|---------|---------|------|------|
| 对话接口 | `/consoleapi/v1/chat` | POST | 统一对话接口，支持流式和阻塞模式 |
| 对话管理 | `/consoleapi/v1/conversations` | GET | 获取对话记录列表 |
| 对话管理 | `/consoleapi/v1/conversations/{id}/messages` | GET | 获取指定对话的消息列表 |

### 1.1 V1 API Token 认证

对话类接口（`/consoleapi/v1/*`）使用双重认证机制：

| 请求头 | 必填 | 说明 |
|--------|------|------|
| `Authorization` | 是 | `Bearer {api_key}`，固定值 |
| `x-sso-token` | 是 | 第三方 JWT Token（通过 `/consoleapi/third-party/auth` 获取） |
| `Content-Type` | 是 | `application/json`，固定值 |

**特殊说明**：

- x-sso-token测试时可以使用的固定值：rhzn_api_test_token
- `/consoleapi/third-party/auth`待完善，后续提供

## 三、接口详情

### 3.1 统一对话接口

**接口**：`POST /consoleapi/v1/chat`

**认证方式**：V1 API Token 认证

**请求头**：

| 参数名 | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | Bearer {api_key} |
| x-sso-token | 是 | 第三方 JWT Token |
| Content-Type | 是 | application/json |

**请求体**：

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| responseMode | string | 否 | "streaming" | 响应模式：`streaming`（流式）或 `blocking`（阻塞） |
| messages | array | 是 | - | 消息列表，标准 AI 对话格式 |
| messages[].role | string | 是 | - | 消息角色：`user` / `assistant` / `system` |
| messages[].content | string/object | 否 | - | 消息内容，支持字符串或数组格式 |
| messages[].attachments | array | 否 | [] | 附件信息列表 |
| messages[].metadata | object | 否 | - | 消息元数据 |
| conversationId | string(UUID) | 否 | - | 对话记录 ID，不传则创建新对话 |
| traceId | string | 否 | - | 链路追踪 ID |
| files | array | 否 | - | 文件列表，`[{ type, url }]` |
| thirdPartyIntegration | object | 否 | - | 第三方平台集成配置 |
| mcpServers | array | 否 | - | MCP 服务器 ID 列表 |
| formVariables | object | 否 | - | 表单变量（用于角色设定变量替换） |
| saveConversation | boolean | 否 | true | 是否保存对话记录 |
| title | string | 否 | - | 对话标题（创建新对话时使用） |
| includeContext | boolean | 否 | - | 是否包含上下文 |
| includeReferences | boolean | 否 | - | 是否返回引用来源 |
| generateSuggestions | boolean | 否 | - | 是否生成自动追问 |
| formFields | array | 否 | - | 表单字段配置 |
| formFieldsInputs | object | 否 | - | 表单字段输入值 |
| billingMode | string | 否 | - | 扣费模式：`creator` / `user` / `all` |
| rolePrompt | string | 否 | - | 角色设定 |
| showContext | boolean | 否 | - | 是否显示对话上下文 |
| showReference | boolean | 否 | - | 是否显示引用来源 |
| enableFeedback | boolean | 否 | - | 是否允许反馈 |
| enableWebSearch | boolean | 否 | - | 是否开启联网搜索 |
| modelConfig | object | 否 | - | 模型配置，`{ id, options }` |
| modelConfig.id | string | 否 | - | 模型 ID |
| modelConfig.options | object | 否 | - | 模型参数：temperature、maxTokens、topP 等 |
| billingConfig | object | 否 | - | 计费配置，`{ price }` |
| datasetIds | array | 否 | - | 知识库 ID 列表（UUID 数组） |
| autoQuestions | object | 否 | - | 自动追问配置，`{ enabled, customRuleEnabled, customRule }` |
| quickCommands | array | 否 | - | 快捷指令配置列表 |
| mcpServerIds | array | 否 | - | MCP 服务器 ID 列表 |

**请求示例**：

```bash
curl -X POST http://localhost:4090/consoleapi/v1/chat \
  -H "Authorization: Bearer {api_key}" \
  -H "x-sso-token: {sso_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "responseMode": "streaming",
    "messages": [
      { "role": "user", "content": "你好" }
    ],
    "conversationId": null,
    "includeReferences": false,
    "generateSuggestions": false
  }'
```

#### 流式响应（SSE）

当 `responseMode=streaming` 时，返回 `text/event-stream` 格式：

```
data: {"type":"references","data":[{...}]}

data: {"type":"conversation_id","data":"xxx-xxx"}

data: {"type":"chunk","data":"你好"}

data: {"type":"chunk","data":"！"}

data: {"type":"reasoning","data":"思考内容..."}

data: {"type":"suggestions","data":["问题1","问题2"]}

data: {"type":"context","data":[...]}

data: [DONE]
```

**事件类型说明**：

| 事件类型 | 说明 |
|---------|------|
| `references` | 知识库引用来源（仅当启用时） |
| `conversation_id` | 新创建对话的 ID |
| `chunk` | AI 回复内容片段 |
| `reasoning` | 推理/思考过程内容（如 DeepSeek） |
| `mcp_tool_detected` | MCP 工具调用检测 |
| `mcp_tool_result` | MCP 工具执行结果 |
| `mcp_tool_error` | MCP 工具执行错误 |
| `tool_calls_completed` | 工具调用完成，继续生成回复 |
| `suggestions` | 自动追问建议 |
| `context` | 对话上下文（仅当启用时） |
| `error` | 错误信息，`{ message, code }` |
| `[DONE]` | 流结束标记 |

#### 阻塞响应（JSON）

当 `responseMode=blocking` 时，返回 JSON：

```json
{
  "conversationId": "uuid...",
  "response": "AI 回复内容",
  "referenceSources": [
    {
      "datasetId": "uuid...",
      "datasetName": "知识库名称",
      "chunks": [
        {
          "id": "uuid...",
          "content": "片段内容",
          "score": 0.95,
          "metadata": {},
          "fileName": "文件名",
          "chunkIndex": 0
        }
      ]
    }
  ],
  "context": [
    { "role": "user", "content": "你好", "timestamp": "2024-01-01T00:00:00.000Z" },
    { "role": "assistant", "content": "你好！", "timestamp": "2024-01-01T00:00:01.000Z" }
  ],
  "suggestions": ["问题1", "问题2"],
  "tokenUsage": {
    "totalTokens": 100,
    "promptTokens": 50,
    "completionTokens": 50
  },
  "responseTime": 1500,
  "annotations": {
    "annotationId": "uuid...",
    "question": "标注问题",
    "similarity": 0.98,
    "createdBy": "用户ID"
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| conversationId | string | 对话记录 ID |
| response | string | AI 完整回复内容 |
| referenceSources | array | 引用来源列表（仅当启用时） |
| context | array | 对话上下文（仅当启用时） |
| suggestions | array | 自动追问建议（仅当启用时） |
| tokenUsage | object | Token 使用统计 |
| tokenUsage.totalTokens | number | 总 Token 数 |
| tokenUsage.promptTokens | number | 提示 Token 数 |
| tokenUsage.completionTokens | number | 补全 Token 数 |
| responseTime | number | 响应时间（毫秒） |
| annotations | object | 标注命中信息（如果匹配到标注） |

**错误响应**：

```json
// 缺少 API 密钥
{ "statusCode": 401, "message": "缺少API密钥" }

// API 密钥无效
{ "statusCode": 401, "message": "API密钥无效或智能体未发布" }

// 缺少第三方 Token
{ "statusCode": 401, "message": "缺少Token" }

// Token 无效或过期
{ "statusCode": 401, "message": "Token无效或已过期" }

// 智能体需要配置模型
{ "statusCode": 400, "message": "智能体需要配置模型" }

// 模型未找到
{ "statusCode": 404, "message": "Model not found." }

// 积分不足
{ "statusCode": 406, "message": "积分不足，请充值" }
```

### 3.2 获取对话记录列表

**接口**：`GET /consoleapi/v1/conversations`

**认证方式**：V1 API Token 认证

**请求头**：

| 参数名 | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | Bearer {api_key} |
| x-sso-token | 是 | 第三方 JWT Token |

**查询参数**：

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| page | number | 否 | 1 | 页码，从 1 开始 |
| pageSize | number | 否 | 15 | 每页条数 |

**请求示例**：

```bash
curl -X GET "http://localhost:4090/consoleapi/v1/conversations?page=1&pageSize=15" \
  -H "Authorization: Bearer {api_key}" \
  -H "x-sso-token: {sso_token}"
```

**响应示例**：

```json
{
  "items": [
    {
      "id": "uuid...",
      "title": "对话标题",
      "userId": "uuid...",
      "modelId": "uuid...",
      "summary": "对话摘要",
      "messageCount": 10,
      "totalTokens": 500,
      "totalPower": 0,
      "config": {},
      "status": "active",
      "isPinned": false,
      "isDeleted": false,
      "metadata": {},
      "createdAt": "2024-01-01T00:00:00.000Z",
      "updatedAt": "2024-01-01T00:00:00.000Z"
    }
  ],
  "total": 100,
  "page": 1,
  "pageSize": 15,
  "totalPages": 7
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| items | array | 对话记录列表 |
| items[].id | string(UUID) | 对话 ID |
| items[].title | string | 对话标题 |
| items[].userId | string(UUID) | 用户 ID |
| items[].modelId | string(UUID) | AI 模型 ID |
| items[].summary | string | 对话摘要 |
| items[].messageCount | number | 消息总数 |
| items[].totalTokens | number | 总 Token 消耗 |
| items[].totalPower | number | 总 Power 消耗 |
| items[].config | object | 对话配置 |
| items[].status | string | 状态：`active` / `completed` / `failed` |
| items[].isPinned | boolean | 是否置顶 |
| items[].isDeleted | boolean | 是否已删除 |
| items[].metadata | object | 扩展数据 |
| items[].createdAt | string | 创建时间 |
| items[].updatedAt | string | 更新时间 |
| total | number | 总记录数 |
| page | number | 当前页码 |
| pageSize | number | 每页条数 |
| totalPages | number | 总页数 |

**业务逻辑**：
- 传入第三方 Token 时，根据第三方用户 ID 和智能体 ID 查询
- 仅传入 API Key 时，根据 API Key 匿名标识查询
- 按 `updatedAt` 和 `createdAt` 降序排列

### 3.3 获取对话消息列表

**接口**：`GET /consoleapi/v1/conversations/{conversationId}/messages`

**认证方式**：V1 API Token 认证

**请求头**：

| 参数名 | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | Bearer {api_key} |
| x-sso-token | 是 | 第三方 JWT Token |

**路径参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| conversationId | string(UUID) | 是 | 对话记录 ID |

**查询参数**：

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| page | number | 否 | 1 | 页码，从 1 开始 |
| pageSize | number | 否 | 15 | 每页条数 |

**请求示例**：

```bash
curl -X GET "http://localhost:4090/consoleapi/v1/conversations/{conversationId}/messages?page=1&pageSize=15" \
  -H "Authorization: Bearer {api_key}" \
  -H "x-sso-token: {sso_token}"
```

**响应示例**：

```json
{

      "id": "uuid...",
      "conversationId": "uuid...",
```

- 先验证对话记录是否属于当前 API Key / 第三方用户
- 无权访问时返回 404
- 消息按 `createdAt` 升序排列

**错误响应**：

```json
// 对话不存在或无权访问
{ "statusCode": 404, "message": "对话记录不存在或无权访问" }
```

## 五、错误码说明

| 状态码 | 错误信息 | 说明 |
|--------|---------|------|
| 400 | 智能体需要配置模型 | 模型未配置 |
| 401 | 缺少API密钥 | 未传 Authorization |
| 401 | API密钥无效或智能体未发布 | API Key 错误 |
| 401 | 缺少Token | 未传 x-sso-token |
| 401 | Token无效或已过期 | JWT 验证失败 |
| 403 | 第三方用户已被禁用 | 用户状态异常 |
| 403 | 关联的平台用户已被禁用 | 平台用户状态异常 |
| 404 | Model not found. | 模型不存在 |
| 404 | 对话记录不存在或无权访问 | 对话不存在或无权限 |
| 406 | 积分不足，请充值 | 余额不足 |
